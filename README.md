## Overview

This repository contains the full simulation and figure-generation code for the paper:

**"Hybrid Optical–Acoustic Communication Protocol for Energy-Efficient High-Data-Rate Underwater IoT Networks with Adaptive Switching"**  
*Kajal Meshram, Dr. S. Gopikrishnan — VIT-AP University, Amaravathi, India*

Underwater IoT networks face a fundamental tension: **optical links** offer Gbps speeds but die at ~100 m, while **acoustic links** reach 5 km but crawl at ~100 Kbps. This work proposes a **Q-Learning agent** that intelligently switches between—or combines—both modalities in real time, based on distance, residual energy, turbidity, and packet size.

## Channel Models

### Optical Channel — Beer-Lambert Law
```
h(d) = exp(−c · d)
```

| Water Type | Attenuation (c) | Max Range |
|---|---|---|
| Pure Sea Water | 0.005 m⁻¹ | ~100 m |
| Clear Ocean | 0.114 m⁻¹ | ~80 m |
| Coastal Water | 0.179 m⁻¹ | ~50 m |
| Turbid/Harbour | 2.190 m⁻¹ | ~15 m |

Peak data rate: **10 Gbps** | Latency: **nanoseconds**

### Acoustic Channel — Thorp's Formula
```
PL(dB) = k·10·log₁₀(d) + α(f)·d/1000

α(f) = 0.11f²/(1+f²) + 44f²/(4100+f²) + 2.75×10⁻⁴f² + 0.003
```
Operating at **25 kHz** | Max range: **5 km** | Peak data rate: **100 Kbps** | Latency: **seconds**

## Q-Learning Agent

### State Space
```python
s = (distance_bin, energy_bin, data_size_bin, turbidity_bin,
     optical_feasible, acoustic_feasible)
```

### Action Space
| Action | Mode | Best for |
|---|---|---|
| 0 | Optical only | Short range ≤ 80 m, clear water, high throughput |
| 1 | Acoustic only | Long range > 500 m, turbid water, low energy |
| 2 | Hybrid (70% opt + 30% aco) | Medium range, balanced load |

### Reward Function
```
R = 0.35·throughput_norm + 0.30·energy_efficiency
    − 0.20·delay_norm + 0.15·PDR − infeasibility_penalty
```

### Hyperparameters

| Parameter | Value |
|---|---|
| Learning rate α | 0.10 |
| Discount factor γ | 0.95 |
| Initial ε | 1.00 → 0.05 |
| ε decay per episode | 0.992 |
| Training episodes | 300 |
| Convergence episode | ~150 |
| States visited | 64 |
